# Day 1 配套示例代码运行说明

这份说明对应 Day 1 的两份示例代码：

- `示例代码/01_环境验证.py`
- `示例代码/02_第一个Selenium脚本.py`

脚本默认使用 `Chrome`。如果你的机器上没有标准 Chrome，或者驱动下载比较慢，也可以临时切到：

- `SELENIUM_BROWSER=edge`
- `SELENIUM_BROWSER=safari`

## 1. 先启动本地练习站

进入练习站目录：

```bash
cd '/Users/Apple/Documents/Codes/PythonTestingProjects/练习项目/Selenium与Appium本地练习站'
```

如果还没有依赖，先安装：

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -U pip
pip install -r requirements.txt
```

启动本地练习站：

```bash
python app/app.py
```

启动后你应该看到：

```text
Selenium 与 Appium 本地练习站已启动: http://127.0.0.1:8000
```

可选验证：

```bash
curl -s http://127.0.0.1:8000/health
```

## 3. 如果运行失败，优先检查

- 本地练习站是否还在运行
- 当前终端是否激活了 `.venv`
- 是否已安装 `selenium`
- 本机浏览器是否可正常启动
- 如果 Chrome 不可用，先切到 `Edge` 或 `Safari`

## 4. Day 1 的正确练法

顺序建议固定为：

1. 先运行 `01_环境验证.py`
2. 确认浏览器和页面都正常
3. 再运行 `02_第一个Selenium脚本.py`
4. 最后自己不看示例代码，重新手敲一遍
