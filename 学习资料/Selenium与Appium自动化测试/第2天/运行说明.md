## 2. 再运行 Day 2 示例代码

新开一个终端：

```bash
cd '/Users/Apple/Documents/Codes/PythonTestingProjects/练习项目/Selenium与Appium本地练习站'
source .venv/bin/activate
```

### 2.1 元素定位练习

```bash
SELENIUM_BROWSER=edge SELENIUM_HEADLESS=1 python '/Users/Apple/Documents/Codes/PythonTestingProjects/学习资料/Selenium与Appium自动化测试/第2天/示例代码/01_元素定位练习.py'
```

预期结果：

- 能打印标题、段落、链接文本、列表项文本
- 说明 `id`、`css`、`link text`、`class name`、`xpath` 都能练到

### 2.2 页面交互练习

```bash
SELENIUM_BROWSER=edge SELENIUM_HEADLESS=1 python '/Users/Apple/Documents/Codes/PythonTestingProjects/学习资料/Selenium与Appium自动化测试/第2天/示例代码/02_页面交互练习.py'
```

预期结果：

- 能打印填充后的用户名
- 能打印提交结果
- 能打印清空后的用户名
- 能打印属性状态文本
- 能打印当前下拉框选中的文本和值

### 2.3 属性获取练习

```bash
SELENIUM_BROWSER=edge SELENIUM_HEADLESS=1 python '/Users/Apple/Documents/Codes/PythonTestingProjects/学习资料/Selenium与Appium自动化测试/第2天/示例代码/03_属性获取练习.py'
```

预期结果：

- 能打印 `id`、`name`、`type`、`placeholder`
- 能打印 `class`、`data-testid`、`href`
- 能打印复选框点击前后的勾选状态

## 3. 如果运行失败，优先检查

- 本地练习站是否还在运行
- 是否已经激活 `.venv`
- 浏览器是否可正常启动
- 如果默认 Chrome 不稳，先切到 `SELENIUM_BROWSER=edge`
