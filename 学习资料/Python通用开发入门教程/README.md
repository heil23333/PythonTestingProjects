# Python 通用开发入门教程

## 1. 教程定位

这是一套面向零基础或弱基础学习者的 Python 系统教材。

它不依赖测试背景，也不要求你提前学过其他编程语言。整套内容按“先学语法，再学组织代码，再学工程化，再做项目”的顺序展开，目标是让你从看不懂 Python，到能独立写出结构清晰的小程序。

这套教程适合下面几类读者：

- 想系统入门 Python
- 学过一点零散语法，但知识不成体系
- 能照着写代码，但还不会独立完成小项目
- 后续想转 Web、数据分析、自动化脚本、测试开发

---

## 2. 这套教程会带你学到什么

学完整套教程后，你应该具备这些能力：

- 能理解 Python 程序是怎样运行的
- 能熟练使用变量、数据类型、流程控制、函数
- 能读写文件、处理异常、拆分模块
- 能写简单类，理解面向对象的基本思想
- 能使用常见标准库解决实际问题
- 能使用虚拟环境和 `pip` 管理依赖
- 能组织一个结构清晰的小型 Python 项目
- 能独立完成中小型练习项目

---

## 3. 学习建议

为了让学习效果更稳定，建议按下面的方式使用这套教程：

1. 每学一小节就亲手敲一次代码，不要只看。
2. 每章都完成“最小可运行示例”和“章节练习题”。
3. 出现报错时，先读报错信息，再回看示例，而不是立刻放弃。
4. 不要追求一次学会全部内容，允许自己先理解 60%，后续反复巩固。
