# 第 3 章：变量、输入输出与基础数据类型

## 1. 这一章学什么

从这一章开始，我们真正进入 Python 代码本身。

无论你以后写什么程序，几乎都绕不开下面几件事：

- 保存数据
- 接收输入
- 输出结果
- 判断数据是什么类型
- 在不同类型之间做转换

这些能力，就是本章的核心。

如果把程序理解成“处理信息的过程”，那么变量和数据类型就是信息被保存和组织的方式。

---

## 2. 什么是变量

变量就是给一个值起名字。

这样做的好处是，你后面就不用反复直接写那个值，而是通过名字引用它。

例如：

```python
name = "Alice"
age = 20
score = 95.5
```

这里：

- `name` 是变量名
- `"Alice"` 是变量里保存的值
- `age` 保存整数 `20`
- `score` 保存小数 `95.5`

你可以把变量理解成一个带标签的小盒子：

- 盒子外面写着名字
- 盒子里面放着数据

---

## 3. 为什么程序需要变量

如果没有变量，很多程序几乎没法正常组织。

例如你想记录一个用户的信息：

```python
print("Alice")
print(20)
print("Alice")
print(20)
```

这段代码虽然也能写，但问题很明显：

- 数据重复
- 不容易修改
- 不容易理解含义

如果改成变量：

```python
name = "Alice"
age = 20

print(name)
print(age)
print(name)
print(age)
```

好处立刻就出来了：

- 代码更清楚
- 修改更方便
- 逻辑更容易维护

---

## 4. 变量怎么命名

变量名不是随便写的。

### 4.1 推荐写法

Python 中通常使用“下划线命名法”：

```python
user_name = "Tom"
total_score = 100
is_login = True
```

这样的命名更容易读。

### 4.2 命名建议

变量名最好满足下面几个原则：

- 见名知意
- 不要太短也不要太长
- 尽量用英文或拼音的一致风格
- 不要随意混用大小写风格

例如：

```python
price = 99
student_count = 30
file_path = "/tmp/demo.txt"
```

### 4.3 不推荐写法

```python
a = "Alice"
b = 20
c = True
```

这种写法在临时测试里可以接受，但在正式学习和项目中可读性很差。

### 4.4 不能这样命名

变量名不能：

- 以数字开头
- 包含空格
- 使用 Python 关键字

错误示例：

```python
# 下面这些写法都不对
2name = "Tom"
user name = "Tom"
if = 10
```

---

## 5. `print()`：把结果输出出来

`print()` 是 Python 初学阶段最常用的函数之一。

它的作用是把内容输出到屏幕上，方便你查看程序运行结果。

例如：

```python
print("你好")
print(123)
print(True)
```

它可以输出很多类型的数据。

### 5.1 输出变量

```python
name = "Alice"
age = 20

print(name)
print(age)
```

### 5.2 一次输出多个内容

```python
name = "Alice"
age = 20

print("姓名：", name, "年龄：", age)
```

这是调试和观察结果时非常常见的写法。

---

## 6. `input()`：接收用户输入

如果说 `print()` 是“输出”，那么 `input()` 就是“输入”。

例如：

```python
name = input("请输入你的名字：")
print("你好，", name)
```

程序执行到 `input()` 时，会暂停等待你输入内容。

你输入完成并按回车后，输入的内容会作为结果返回。

### 6.1 一个非常重要的细节

`input()` 接收到的内容，默认是字符串 `str`。

例如：

```python
age = input("请输入年龄：")
print(age)
print(type(age))
```

即使你输入的是 `18`，它得到的仍然是字符串，而不是整数。

这个细节后面做计算时非常重要。

---

## 7. 什么是数据类型

程序不仅要存值，还要知道“这个值是什么种类”。

例如：

- `"Alice"` 是文字
- `18` 是整数
- `95.5` 是小数
- `True` 是布尔值

不同种类的数据，能做的操作不同。

这就是数据类型存在的原因。

本章重点掌握 4 种基础数据类型：

- `int`：整数
- `float`：浮点数，也就是小数
- `str`：字符串
- `bool`：布尔值

---

## 8. 整数 `int`

整数就是没有小数部分的数字。

例如：

```python
age = 18
year = 2026
count = 100
```

整数常用于表示：

- 年龄
- 数量
- 编号
- 页码

### 8.1 整数的常见运算

```python
a = 10
b = 3

print(a + b)
print(a - b)
print(a * b)
print(a // b)
print(a % b)
```

这里：

- `+` 是加法
- `-` 是减法
- `*` 是乘法
- `//` 是整除
- `%` 是取余

---

## 9. 浮点数 `float`

浮点数就是带小数部分的数字。

例如：

```python
price = 19.9
height = 172.5
weight = 60.8
```

它常用于：

- 价格
- 身高
- 体重
- 平均分

### 9.1 浮点数参与运算

```python
price = 19.9
count = 3

print(price * count)
```

整数和浮点数可以一起运算，结果通常会变成浮点数。

---

## 10. 字符串 `str`

字符串就是文本。

只要是放在引号里的内容，通常就是字符串。

例如：

```python
name = "Alice"
city = 'Shanghai'
message = "欢迎学习 Python"
```

### 10.1 字符串可以包含数字

```python
phone = "13800138000"
```

虽然看起来像数字，但如果它在引号里，它就是字符串。

### 10.2 字符串拼接

```python
first_name = "Li"
last_name = "Lei"

print(first_name + last_name)
```

### 10.3 字符串重复

```python
print("ha" * 3)
```

输出结果是：

```text
hahaha
```

---

## 11. 布尔值 `bool`

布尔值只有两个：

- `True`
- `False`

它通常表示：

- 是或否
- 对或错
- 成功或失败
- 开启或关闭

例如：

```python
is_student = True
is_login = False
```

布尔值经常和条件判断一起使用，后面学习 `if` 时会大量用到。

---

## 12. 用 `type()` 查看数据类型

当你不确定一个值是什么类型时，可以用 `type()`。

例如：

```python
name = "Alice"
age = 20
price = 19.9
is_ok = True

print(type(name))
print(type(age))
print(type(price))
print(type(is_ok))
```

这对于初学阶段特别有用，因为很多错误都和“类型理解错了”有关。

---

## 13. 类型转换为什么重要

程序里经常会遇到这样的情况：

- 用户输入的是字符串
- 你却想拿它做数字计算

例如：

```python
age = input("请输入年龄：")
print(age + 1)
```

这段代码会报错，因为 `age` 是字符串，不能直接和整数 `1` 相加。

所以我们需要类型转换。

---

## 14. 常见类型转换

### 14.1 转成整数 `int()`

```python
age = input("请输入年龄：")
age = int(age)
print(age + 1)
```

### 14.2 转成浮点数 `float()`

```python
price = "19.9"
price = float(price)
print(price)
```

### 14.3 转成字符串 `str()`

```python
age = 20
text = str(age)
print("年龄是：" + text)
```

### 14.4 转成布尔值 `bool()`

初学阶段先知道它存在即可，后面学条件判断时再深入。

```python
print(bool(1))
print(bool(0))
print(bool(""))
```

---

## 15. 一个完整的小示例

下面写一个稍完整的程序，综合使用变量、输入输出和类型转换。

```python
name = input("请输入你的名字：")
age = int(input("请输入你的年龄："))
height = float(input("请输入你的身高（米）："))

print("你的姓名是：", name)
print("明年你将是：", age + 1, "岁")
print("你的身高是：", height, "米")
```

这个程序里发生了几件事：

1. 程序接收姓名输入，并保存到变量 `name`
2. 程序接收年龄输入，并用 `int()` 转成整数
3. 程序接收身高输入，并用 `float()` 转成浮点数
4. 最后把处理后的结果输出出来

这已经是一个很完整的入门程序了。

## 16. 为什么 `input()` 得到的是字符串

这个点值得单独再强调一次。

当用户在键盘输入内容时，程序拿到的首先是“文本形式”的信息。

例如你输入：

```text
18
```

程序最先接收到的是字符 `"1"` 和 `"8"` 组成的文本，所以默认结果是字符串。

这也是为什么你需要在做数学计算前主动写：

```python
age = int(input("请输入年龄："))
```

## 17. 常见错误示例

### 17.1 把字符串和数字直接相加

错误示例：

```python
age = input("请输入年龄：")
print(age + 1)
```

原因：

- `age` 是字符串
- `1` 是整数
- 两种不同类型不能这样直接相加

### 17.2 变量名起得没有意义

错误示例：

```python
a = "Tom"
b = 18
c = 99.5
```

虽然程序也能跑，但别人很难快速看懂。

### 17.3 以为引号里的数字还是数字

```python
num = "100"
```

这里的 `num` 是字符串，不是整数。

### 17.4 用户输入无法转成整数

例如：

```python
age = int(input("请输入年龄："))
```

如果用户输入的是：

```text
abc
```

程序就会报错，因为 `"abc"` 不能转成整数。

这个问题后面可以通过异常处理优化。
